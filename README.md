## Running the project

Simply run

```console
python main.py
```
OR

```console
python3 main.py
```

## For using pipenv

Run this command below to activate or initialize a pipenv virtual environment
```console
pipenv shell
```

This will require python version 3.11

If you are not running python 3.11, then don't need this
